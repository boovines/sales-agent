## Codebase Patterns
- Use `from __future__ import annotations` at top of all files with type hints (Python 3.9.6 compat)
- LangGraph MemorySaver import: `from langgraph.checkpoint.memory import MemorySaver`
- LangGraph Command import: `from langgraph.types import Command`
- Never modify `adela_outbound/api/main.py` or `adela_outbound/scheduler.py` — owned by feature/agent-discovery
- Never modify `adela_outbound/db/schemas.py` or `adela_outbound/db/contracts.py` — owned by feature/agent-discovery
- Config singleton: `from adela_outbound.config import config`
- DB connection: `from adela_outbound.db.connection import get_db` used as `async with get_db() as conn:`
- pytest with `asyncio_mode = auto` in pytest.ini — no need for `@pytest.mark.asyncio` decorator
